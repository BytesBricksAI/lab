# SimPlant-Lab rebrand — implementation notes

Living document updated during the rebrand implementation. HTML mirror: [implementation-notes.html](implementation-notes.html).

## Decisions not in the original plan

| Decision | Rationale |
|----------|-----------|
| `DEFAULT_DOCS_URL` = `https://github.com/SimPlant/SimPlant-v2` | No production docs URL was specified; using repo root until a docs site exists. |
| Visual assets (PNG/ICO) unchanged | No SimPlant-Lab artwork supplied; `simplant_lab_wordmark.svg` is a minimal text placeholder. Logos/favicons still show Rerun graphics. |
| `import rerun` shim package retained | `rerun_py/rerun_sdk/rerun/__init__.py` re-exports `simplant_lab` with `DeprecationWarning`. |
| Pixi: `simplant-lab` / `simplant-lab-build` + `rerun` aliases | Both names run `simplant-lab-cli`. |
| `rs-fmt` builds `snippets` first | `docs/snippets/src/snippets/mod.rs` is gitignored and generated by `build.rs`. |
| FlatBuffers `namespace rerun` unchanged | Wire/API compatibility per plan. |
| `rerun-sdk` kept in uv `pyproject.toml` workspace | Listed alongside `simplant-lab-sdk` until all workspace refs are migrated. |
| `rerun_bindings` PyO3 module name unchanged | Renaming would break maturin/FFI without a larger migration. |
| Serde field rename with alias | `include_simplant_lab_examples_button_in_recordings_panel` reads old `include_rerun_examples_*` from saved viewer state. |

## Concessions / scope limits

- `pixi run codegen` and `pixi run man` were **not** executed (long build; templates updated manually for `"into SimPlant-Lab"`).
- `docs/content/reference/types/**` and `docs/content/reference/cli.md` not regenerated.
- Many `https://rerun.io/...` links remain in intro cards, view `.docs_link()`, and help text.
- `examples/python` / `tests/python` / `docs/snippets` received bulk `import simplant_lab` — review edge cases.
- `cargo check -p simplant-lab-cli` not completed in CI here (NASM permission in environment); Cargo manifest parsing errors were fixed (`simplant-lab/sdk` feature path).
- C++ `namespace rerun::` and CMake target `rerun_sdk` left as-is (plan scope).
- `CHANGELOG.md` history not rewritten (upstream Rerun changelog preserved).
