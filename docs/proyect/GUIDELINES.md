# SimPlant Lab 

Decidimos forkear [Rerun.io](Rerun) porque proporciona las herramientas adecuadas para crear SimPlant Lab. SimPlant Lab tiene como objetivo proporcionar las herramientas necesarias para manipular los datos que las instalaciones de procesamiento de pretróleo de la industria Oil & Gas.

La industria Oil & Gas se está transformando con la implementación y la utilización de la inteligencia artificial tanto para optimizar procesos como para mejorar la seguridad de los mismos. Desde ByteBricks queremos liderar este cambio con SimPlant, brindando todos los recursos necesarios y adecuados para que las empresas puedan tener la mejor experiencia posible en este cambio paradigmático que esta viviendo esta industria.

Dicho esto, queremos crear SimPlant Lab utilizando como base Rerun. SimPlant Lab es la capa de datos para IA de la industria Oil & Gas. Suministramos las primitivas de datos para construir, comprender y mejorar sus bucle de datos. Esta diseñada para datos multimodales y de múltiples velocidades, desde la primera grabación hasta la escala masiva. Así mismo debe permitir el entranamiento y en tiempo real de modelos de IA, simulaciones de procesos, tests de estrés de equipos y cañerías.


## Tarea

Tu tarea es diseñar un plan de migración para logar los objetivos propuestos.
Vamos a utilizar arquitectura hexagonal aplicando el patrón Agreggate, y diviendo cada módulo en las capas de Dominio, Presentación, Aplicación e Infraestructura cuando sea necesario.
